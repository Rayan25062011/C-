## Why
C- is a language heavily inspired by C++, python, and java, but simpler.

# Why should you use it
Tired of C? Use ***C-***!

## Get Started
> git clone https://github.com/Rayan25062011/C-
> 
> chmod +x src
> 
> cd src
> 
> python3 interpreter.py -r example.cm
